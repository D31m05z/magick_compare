-----
usage
-----
magick_compare.sh [-a] [-b] [-s n] -- You must be provided two folder path for comparing the images and the start frame index.

where:
    -h  show this help text
    -a  folder1
    -b  folder2
    -s  set the start frame

-------
example
-------
./magick_compare.sh -a ./folder2/ -b ./folder1/ -s 10
